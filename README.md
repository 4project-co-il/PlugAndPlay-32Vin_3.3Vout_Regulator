# PlugAndPlay-32Vin_3.3Vout_Regulator
# PlugAndPlay: Regulator module for 32V IN, output: 3.3V/2A
This is the repository with the files for PlugAndPlay regulator module that accepts up to 32V input and output 3.3V/2A.

This module allows connecting higher voltages to the Plug-n-Play development boards.
